## Configuration

**Environment:**
- Single environment variable: `VITE_NETWORK` (defaulted to "MAINNET")
- Network configuration in `src/lib/networks.ts` - defines RPC/indexer URLs per network
- `.env` file present - contains `VITE_NETWORK=MAINNET`

**Build:**
- Vite configuration with plugin chain: tailwindcss → tanstackRouter → viteReact
- Manual chunk splitting in rollupOptions:
  - `vendor-react`: React core
  - `vendor-tanstack`: TanStack Query, Router, Form
  - `vendor-utils`: Utility libraries
  - `vendor-aptos`: Blockchain SDK and Surf
- Chunk size warning limit: 1000KB
- TypeScript path alias: `@/*` → `./src/*`
- GraphQL schema plugin for IDE support
